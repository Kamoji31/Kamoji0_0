Array

An array is a collection of elements, typically of the same type, stored in contiguous memory locations. It allows efficient access to elements via indices, making it a commonly used data structure in programming. Arrays are widely used for storing data in sequential formats, such as lists, matrices, and even tables.

Importance of Arrays

Arrays play a significant role in computer science and programming for several reasons:

  • Efficient Access: Arrays provide quick access to elements using indices, allowing constant time complexity for lookups.
  • Memory Efficiency: Arrays use a contiguous block of memory, making them more memory efficient compared to other data structures like linked lists.
  • Organized Data: Arrays help in organizing data efficiently and are ideal for situations where elements need to be processed in sequence.
  • Basic Data Structure: Arrays serve as the foundational building block for more complex data structures like matrices, heaps, and hash tables.

Common Operations on Arrays

  • Insertion: Adding an element at a specific index or at the end of the array.
  • Deletion: Removing an element from a specific index or from the end of the array.
  • Traversal: Accessing each element in the array, often used for displaying or performing operations on all elements.
  • Search: Finding the index of a specific element in the array.

Sorting Algorithms

Sorting is a fundamental operation that involves arranging elements in a specific order, typically ascending or descending. Here are 5 common sorting algorithms implemented in Python:

1. Bubble Sort

Bubble Sort is a simple comparison-based algorithm that repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order. The process repeats until the array is sorted.

a = [64, 34, 25, 12, 22, 11, 90]

for i in range(len(a)):
    for j in range(0, len(a)-i-1):
        if a[j] > a[j+1]:
            a[j], a[j+1] = a[j+1], a[j]

print(a)  # Output will be [11, 12, 22, 25, 34, 64, 90]
Output
[11, 12, 22, 25, 34, 64, 90]
2. Selection Sort

Selection Sort works by repeatedly selecting the smallest (or largest) element from the unsorted part of the array and swapping it with the element at the current position.

a = [64, 25, 12, 22, 11]

for i in range(len(a)):
    min_index = i
    for j in range(i+1, len(a)):
        if a[j] < a[min_index]:
            min_index = j
    a[i], a[min_index] = a[min_index], a[i]

print(a)  # Output will be [11, 12, 22, 25, 64]
Output
[11, 12, 22, 25, 64]
3. Insertion Sort

Insertion Sort builds the final sorted array one item at a time. It assumes that the first element is already sorted, then inserts each subsequent element into its correct position in the sorted portion.

a = [12, 11, 13, 5, 6]

for i in range(1, len(a)):
    key = a[i]
    j = i-1
    while j >= 0 and key < a[j]:
        a[j+1] = a[j]
        j -= 1
    a[j+1] = key

print(a)  # Output will be [5, 6, 11, 12, 13]
Output
[5, 6, 11, 12, 13]
4. Merge Sort

Merge Sort is a divide-and-conquer algorithm that divides the array into halves, recursively sorts each half, and then merges the sorted halves to produce the final sorted array.

def merge_sort(a):
    if len(a) > 1:
        mid = len(a)//2
        left_half = a[:mid]
        right_half = a[mid:]

        merge_sort(left_half)
        merge_sort(right_half)

        i = j = k = 0
        while i < len(left_half) and j < len(right_half):
            if left_half[i] < right_half[j]:
                a[k] = left_half[i]
                i += 1
            else:
                a[k] = right_half[j]
                j += 1
            k += 1

        while i < len(left_half):
            a[k] = left_half[i]
            i += 1
            k += 1

        while j < len(right_half):
            a[k] = right_half[j]
            j += 1
            k += 1

a = [38, 27, 43, 3, 9, 82, 10]
merge_sort(a)
print(a)  # Output will be [3, 9, 10, 27, 38, 43, 82]
Output
[3, 9, 10, 27, 38, 43, 82]
5. Quick Sort

Quick Sort is a divide-and-conquer algorithm that picks a pivot element, partitions the array into two sub-arrays (elements less than the pivot and elements greater than the pivot), and recursively sorts the sub-arrays.

def quick_sort(a):
    if len(a) <= 1:
        return a
    pivot = a[len(a) // 2]
    left = [x for x in a if x < pivot]
    middle = [x for x in a if x == pivot]
    right = [x for x in a if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)

a = [3, 6, 8, 10, 1, 2, 1]
print(quick_sort(a))  # Output will be [1, 1, 2, 3, 6, 8, 10]
Output
[1, 1, 2, 3, 6, 8, 10]
