Queues

A Queue is a linear data structure that follows the First In First Out (FIFO) principle. The first element added to the queue will be the first one to be removed. Queues are used in scenarios where order needs to be maintained, such as scheduling tasks, managing buffers, or handling requests in systems.

Importance of Queues

Queues play a significant role in computer science and programming for several reasons:

  • FIFO Principle: The queue operates on the First In First Out (FIFO) principle, ensuring that elements are processed in the order they arrive.
  • Task Scheduling: Queues are used to manage tasks that need to be executed in a specific order, such as print jobs or requests to a server.
  • Memory Management: Queues are used in operating systems for managing memory and ensuring that processes are handled in the correct order.
  • Event Handling: In event-driven systems, queues help manage events that occur asynchronously, ensuring proper sequencing and execution.

Common Operations on Queues

  • Enqueue: Adding an element to the rear (end) of the queue.
  • Dequeue: Removing an element from the front of the queue.
  • Front: Accessing the element at the front of the queue without removing it.
  • IsEmpty: Checking if the queue is empty.
  • Size: Returning the number of elements in the queue.

Types of Queues

Queues come in different variations, each suited to specific use cases:

1. Simple Queue

In a simple queue, elements are added at the rear and removed from the front, following the FIFO (First In, First Out) principle. It is the most basic type of queue.

Example:
class SimpleQueue:
    def __init__(self):
        self.queue = []

    def enqueue(self, item):
        self.queue.append(item)

    def dequeue(self):
        if self.is_empty():
            return "Queue is empty"
        return self.queue.pop(0)

    def is_empty(self):
        return len(self.queue) == 0

# Example usage
queue = SimpleQueue()
queue.enqueue(5)
queue.enqueue(10)
queue.enqueue(15)
print(queue.dequeue())  # Output: 5
print(queue.queue)  # Output: [10, 15]
Output
Dequeue element: 5
Queue after dequeue: [10, 15]
2. Circular Queue

A circular queue connects the rear of the queue to the front, making efficient use of memory by allowing the rear pointer to wrap around when it reaches the end of the queue.

Example:
class CircularQueue:
    def __init__(self, size):
        self.queue = [None] * size
        self.front = self.rear = -1
        self.size = size

    def enqueue(self, item):
        if (self.rear + 1) % self.size == self.front:
            return "Queue is full"
        if self.front == -1:
            self.front = self.rear = 0
        else:
            self.rear = (self.rear + 1) % self.size
        self.queue[self.rear] = item

    def dequeue(self):
        if self.front == -1:
            return "Queue is empty"
        removed = self.queue[self.front]
        if self.front == self.rear:
            self.front = self.rear = -1
        else:
            self.front = (self.front + 1) % self.size
        return removed

# Example usage
queue = CircularQueue(3)
queue.enqueue(5)
queue.enqueue(10)
queue.enqueue(15)
print(queue.dequeue())  # Output: 5
queue.enqueue(20)  # The rear pointer wraps around
print(queue.queue)  # Output: [20, 10, 15]
Output
Dequeue element: 5
Queue after wrapping around: [20, 10, 15]
3. Priority Queue

In a priority queue, each element is assigned a priority. Elements with higher priority are dequeued before elements with lower priority, regardless of their position in the queue.

Example:
import heapq

class PriorityQueue:
    def __init__(self):
        self.queue = []
        self.index = 0

    def enqueue(self, priority, item):
        heapq.heappush(self.queue, (-priority, self.index, item))
        self.index += 1

    def dequeue(self):
        if not self.queue:
            return "Queue is empty"
        return heapq.heappop(self.queue)[-1]

# Example usage
queue = PriorityQueue()
queue.enqueue(2, 'task 1')
queue.enqueue(1, 'task 2')
queue.enqueue(3, 'task 3')
print(queue.dequeue())  # Output: task 3 (highest priority)
print(queue.dequeue())  # Output: task 1
Output
Dequeue element (highest priority): task 3
Dequeue element: task 1
4. Double-Ended Queue (Deque)

A deque allows insertion and removal of elements from both the front and rear of the queue. This makes it more flexible than a simple queue.

Example:
from collections import deque

# Using Python's deque module
queue = deque()

queue.append(10)  # Enqueue at the rear
queue.append(20)
queue.append(30)

print(queue.popleft())  # Output: 10 (dequeued from front)
queue.appendleft(5)  # Enqueue at the front
print(queue)  # Output: deque([5, 20, 30])
Output for Deque
Dequeue from front: 10
Queue after adding to the front: deque([5, 20, 30])
